Desvendando as TAGS do h1 ao h6

voltar

As tags <h1> até <h6> representam títulos e subtítulos. O <h1> é o mais importante e maior, enquanto o <h6> é o menor e menos relevante.

São usadas para estruturar e hierarquizar o conteúdo de uma página, como capítulos e seções de um livro.

Além de organizarem visualmente, ajudam na acessibilidade e no SEO.

Desvendando as TAGs p; mark; small

voltar

A tag <p> define um parágrafo de texto. Ela é usada para separar blocos de informação escrita.

A tag <mark> serve para destacar parte de um texto, como se fosse um marca-texto. Exemplo: isso está destacado.

A tag <small> exibe o texto em tamanho menor, geralmente usada para observações. Exemplo: nota de rodapé.

Desvendando as TAGS ol; ul

voltar

A tag <ol> cria uma lista ordenada (numerada), e cada item é definido com <li>.

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Já a tag <ul> define uma lista não ordenada (com marcadores), também usando <li> para os itens.

  • Item 1
  • Item 2
  • Item 3

Desvendando as TAGS a; hr

A tag <a> cria links. Por exemplo, clique aqui para visitar a MDN Web Docs.

A tag <hr> cria uma linha horizontal na página. Ela é usada para separar visualmente seções de conteúdo, como uma divisão entre assuntos.
